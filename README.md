# Interactive-Story-Game-in-C

This is a text-based interactive game where the player is given different options to choose from in each stage. Based on the selected options the game will have different paths. There are traps and loopholes in each stage which may make the layer lose the game. Only selecting the correct options will take the player to the Final Stage of the game. The player score is also maintained in the game which depends on the options selected by the player throughout the game. The player's target is to complete the game in the maximum points possible.
